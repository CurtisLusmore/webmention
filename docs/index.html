Webmention

Have you written a response to one of my posts? Then create a Webmention by submitting the below form, sending a POST request to https://webmention.lusmo.re/post (like below), or using publishing software that supports Webmentions. 

curl --request POST 'https://webmention.lusmo.re/post' \
--header 'Content-Type: application/x-www-form-urlencoded' \
--data-urlencode 'source=https://your.blog/post' \
--data-urlencode 'target=https://lusmo.re/webmention'
